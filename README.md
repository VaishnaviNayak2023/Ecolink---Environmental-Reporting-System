# EcoLink - Environmental Incident Reporting System

EcoLink is a web-based platform that allows users to report environmental incidents efficiently. Users can choose between a **Quick Report** for urgent situations or a **Guided Report** for a detailed step-by-step submission. The system is designed to help environmental agencies respond faster and track incidents effectively.

## Features

- **Quick Report:** Submit essential details in under 2 minutes. Ideal for urgent incidents.
- **Guided Report:** Step-by-step form for comprehensive reporting of complex environmental issues.
- **Responsive Design:** Works seamlessly on desktop, tablet, and mobile devices.
- **Secure:** All reports are transmitted securely to ensure user privacy.
- **User Authentication:** Users can sign up or log in to track their submitted reports.
- **Interactive UI:** Hover effects, smooth scrolling, and intuitive navigation for a modern experience.

## Installation

Follow these steps to set up the project locally:

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/ecolink.git
   cd ecolink

2. **Install dependencies**
   ```bash
   npm install
   

4. **Start the development server**
   ```bash
   npm start

5. **Open the project in your browser**
   ```bash
   http://localhost:3000

**Usage**
Quick Report: Click "Quick Report," fill in the essential details, and submit.
Guided Report: Click "Guided Report," follow the step-by-step instructions, and submit.
Track Reports: Log in to view, edit, or delete your submitted reports.

**Technologies**
Frontend: HTML5, CSS3, JavaScript
Backend: PHP
Database: MySQL and XAMPP
Deployment: Vercel
